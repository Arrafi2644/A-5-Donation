FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the page structure as a tree of objects. It allows scripts like JavaScript to access, modify, and manipulate HTML or XML content dynamically. Each element, attribute, and text is represented as a node within the tree

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use methods like document.getElementById('id'), document.querySelector('selector'), or document.getElementsByClassName('class'). These methods allow you to target elements by their ID, class, or CSS selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. It's useful because it improves performance by reducing the number of event listeners and allows dynamic child elements to be handled efficiently. This also simplifies event management.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using JavaScript by selecting the element with document.getElementById() or similar methods, and then modifying attributes with setAttribute() or directly accessing properties like element.src. To change styles, use element.style.propertyName = value or modify class names with classList.
